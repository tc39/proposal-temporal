Amendments to the ECMAScript® 2021 Language Specification

This section lists amendments which must be made to ECMA-262, the ECMAScript® 2021 Language Specification, other than the addition of the new sections specifying the Temporal object and everything related to it. Text to be added is marked like this, and text to be deleted is marked like this.

Properties of the Date Prototype Object

Date.prototype.toTemporalInstant ( )

The following steps are performed:

1. Let _t_ be ? thisTimeValue(*this* value). 1. Let _ns_ be ? NumberToBigInt(_t_) × 106. 1. Return ! CreateTemporalInstant(_ns_).
